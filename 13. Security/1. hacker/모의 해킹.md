#### 와이파이 비밀번호 알기

```bash
//cmd 창에서
$ netsh /?

$ netsh wlan show 
// profiles

$ netsh wlan show profiles
// $ netsh wlan show profiles <무선랜 이름>

//무선랜 정보 나오면 보안키 찾기 위해
$ netsh wlan show profiles /?
//key를 일반 텍스트로 표시하려면 key=clear로 설정

$ netsh wlan show profiles name="<무선랜 이름>" key=clear

// 키 콘첸츠에 무선랜 비밀번호 표시 됨.
```
